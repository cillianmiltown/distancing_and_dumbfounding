Analysis of Simulated Data

The following is a sample analysis using a simulated data set. This simulated data set contains N = 2400 participants. This sample size was chosen in order for at least 200 participants at each level of the temporal distance IV when analysing the scenarios separately (i.e., aiming for n = 600 per scenario). This sample size will also allow for the detection of a medium effect in the combined analysis on the entire sample.

Temporal Distancing and Dumbfounding

Overview of Judgments

One thousand Six hundred ninety participants (70.42%) rated the behavior of Julie and Mark as wrong initially, and one thousand five hundred Ninety eight participants (66.58%) rated the behavior as wrong at the end of the task. There was a significant difference between initial ratings (M = 3.3, SD = 1.3) and revised ratings (M = 3.4, SD = 1.4), t(2399) = -2.54, p = .011, d = 0.05.

Distancing and Judgments Made

There was no difference in initial judgement depending on distance manipulation: F(2, 2397) = 0.68, p = .507 \(\eta\)p2 = 0.001, (Mincreased = 3.3, SDincreased = 1.4, Mdecreased = 3.2, SDdecreased = 1.3, Mcontrol = 3.3, SDcontrol = 1.3). There was no difference in revised judgement depending on distance manipulation: F(2, 2397) = 1.91, p = .148, \(\eta\)p2 = 0.002, (Mincreased = 3.4, SDincreased = 1.4, Mdecreased = 3.4, SDdecreased = 1.4, Mcontrol = 3.3, SDcontrol = 1.3).

Distancing and Reason-Giving/Dumbfounding

There was a significant association between temporal distance condition and response to the critical slide, \(\chi\)2(4, N = 2400) = 51.338, p < .001, V = 0.15, the observed power was 1. The responses to the critical slide for the increased distance group (N = 800) the decreased distance group (N = 800), and the control group (N = 800) are displayed in {apafg-tempoverall}.

where’s the table

Overall the model significantly predicted responses to the critical slide \(\chi\)2(16, N = 2400) = 85.74, p < .001, The observed power was 1. The model explained between 1.18% (Cox and Snell R square) and 2.78% (Nadelkerke R squared) of the variance in responses to the critical slide. For scenarios in the future, participants were more likely to provide reasons than to present as dumbfounded Wald = -5.78, p = .004, odds ratio = 0.47, 95% CI [0.28, 0.78].
